## CocoaPods

For managing libraries in this project I use Cocoapods.

### Cocoapods-Keys

Project uses `cocoapods-keys` plugin to manage secrets for the app.
For more info, check [cocoapods-keys](https://github.com/orta/cocoapods-keys).
